Описание на сайта.

Това е сайт за калкулатор, който конвертира лева в ваша по избор валута. За изчисляването съм използвал javascript код. Имаме input box в който въвеждаме колко лева искаме да конвертираме и чрез select box избираме в каква валута да конвертираме.
